# HTML
practice questions on html
Q1.To illustrate the Basic HTML structure.
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" 
          content="width=device-width, initial-scale=1.0">
    <title>First Program</title>

</head>

<body>
    <h>First Heading</h>
    <p>This is my first Paragraph</p>
</body>

</html>
